# Electoral Anomaly Simulator (選挙得票率の異常検知シミュレータ)

## Overview (概要)
本プロジェクトは、選挙やアンケート調査において、ある選択肢（候補者）への得票率の偏りが「自然な揺らぎ」なのか、あるいは「同調圧力、組織票、不正などの異常な圧力」によるものなのかを、**大偏差原理（Large Deviation Theory）**を用いて定量的に検証・可視化するシミュレーションツールです。

カルバック・ライブラー情報量（KL Divergence）を用いて確率の対数減衰（$\log_{10} P$）を計算し、「社会科学的限界」「物理的限界」「計算機的限界」という3つの絶対的な閾値と照らし合わせることで、集団の意思決定プロセスにおける異常性を数学的に証明します。

## Features (主な機能)
* **KLダイバージェンスによる高精度な確率計算**: 得票率がランダムな期待値（$1/M$）から乖離した際に偶然そうなる確率を、浮動小数点の限界まで計算。
* **候補者数ごとの動的シミュレーション**: 候補者数 $M = 2, 3, 4, 5, 10$ のパターンを同時にシミュレート。
* **限界閾値の可視化**:
  * `Social Limit` ($10^{-9}$): 全人類に1回起こる確率。強い同調圧力や強固な地盤の存在を示唆。
  * `Physical Limit` ($10^{-80}$): 宇宙に存在する観測可能な原子の数。物理的な独立試行では到達不可能。
  * `Computational Limit` ($10^{-308}$): Float64アンダーフロー限界。絶対的な虚構・データ改ざんの疑い。
* **一括エクスポート**: シミュレーション結果のグラフ（PNG）と各条件の詳細データ（CSV）を自動的にZIP化して出力。

## Requirements (動作環境)
* Python 3.8+
* NumPy
* Pandas
* Matplotlib

インストール手順:
```bash
pip install -r requirements.txt

python simulate_anomaly.py

Outputs (出力ファイル)実行後、カレントディレクトリに electoral_anomaly_simulation.zip が生成されます。解凍すると以下のファイルが含まれています。electoral_anomaly_plot.png (限界閾値と確率減衰のグラフ)electoral_anomaly_M2.csv 〜 M10.csv (候補者数ごとの計算結果データ)Theoretical Background (理論的背景)統計力学および情報理論において、マクロな観測量（得票率 $q$）が平衡状態（期待値 $p=1/M$）から外れる確率は、大偏差原理に基づき以下のようにレート関数（KL情報量）を用いて記述されます。$$ P \approx \exp(-N \cdot D_{KL}(q || p)) $$本シミュレーションは、集団サイズ $N$ が巨大な場合、わずかなパーセンテージの偏りであっても、偶然発生する確率が指数関数的にゼロへ落下する「情報の剛性」を可視化します。これにより、極端な得票率が「人為的なアトラクタ（引力）」なしには存在し得ないことを数理統計学的に実証します。License (ライセンス)This project is licensed under the MIT License - see the LICENSE file for details.
